# fintprosjektet.github.io

* `Master branch` contains static files hosted on [fintprosjektet.github.io](https://fintprosjektet.github.io/)
* `Source branch` contains the hugo project files 

### Run local server
-w = watch  
`hugo server -w`

### Deploy to github pages  
`./deploy`
